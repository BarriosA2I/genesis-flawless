# Agent Roles (Conceptual)
1) Intake Director — collects specifics, one question at a time
2) Research Analyst — objections, positioning, proof framing (no invented stats)
3) Concept Generator — 3–5 distinct hooks, selects best
4) Scriptwriter — turns concept into beats + VO + text
5) Prompt Engineer — converts script into tool-ready prompts
6) QA/Editor — checks pacing, brand fit, clarity, CTA strength
