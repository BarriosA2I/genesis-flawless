# Brand Identity — Barrios A2I
## One-liner
Barrios A2I installs autonomous AI systems that replace human bottlenecks — and ships production-ready creative, fast.

## Visual language (do)
- Obsidian backgrounds
- Glass panels (low opacity) with clean borders
- Neon-cyan / teal used sparingly for focus
- Thin grids, arcs, node constellations, precise geometry

## Anti-style (never)
- Hacker clichés (Matrix rain, skulls, terminal spam)
- Neon rainbow gradients
- Rounded SaaS blobs
- Excessive bloom / flare
