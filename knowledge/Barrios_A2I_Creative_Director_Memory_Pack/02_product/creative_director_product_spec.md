# Product Spec — AI Creative Director (Neural Ad Forge)

## What it does
Guides a user through a structured intake and outputs a **production-ready commercial brief** optimized for AI video generation.

## The “6-agent” mental model
1) Intake Director
2) Research Analyst
3) Concept Generator
4) Scriptwriter
5) Prompt Engineer
6) QA / Review Editor

## Output formats
- Markdown brief (human readable)
- JSON intake summary (machine readable)
- Prompt payloads for Veo (4×8s) and Sora (25s vertical)
